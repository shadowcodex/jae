## Instructions

- Play Music - hit play
- Pause Music - hit pause
- Save File As HTML - hit save
- View Bubble Toolbar - highlight any text
- Open Fullscreen / zen mode - Click fullscreen
- Write A Best Selling Novel - Probably not, but give it a shot anyways!
